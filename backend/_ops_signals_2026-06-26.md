# Сигналы для эксплуатации — КС «Ахангаран», 2026-06-26

> **СТАТУС: ПОБОЧНЫЕ сигналы.** Добыты на нечистых данных в ходе отладки методологии
> v2-CQR (сломанная пара близнецов, разные окна). **НЕ выход валидированного пайплайна.**
> Читать как **«посмотрите вот сюда»**, НЕ как «дефект подтверждён». Оба требуют
> инженерного подтверждения до любого действия; сами по себе — не основание для тревоги/останова.

## Сигнал 1 — ГПА-1: растущий остаток на масляной/газовой температуре

- **Что:** остаток модели на `oil_temp_out_st` (слабее — `gas_temp_out_gpa`) **растёт по ходу
  окна** после 18.06, при том что текущее значение остаётся в физических лимитах (running-гейт
  не срабатывает — поэтому штатный мониторинг это не видит).
- **Измерено** (остаток в σ обучающего шума, первые 60% окна → последние 40%):
  - `oil_temp_out_st`: bias **+1.23σ → +4.57σ**, разброс **×1.4**;
  - `gas_temp_out_gpa`: bias **+0.24σ → +1.91σ**.
- **Локализация:** ГПА-2 в том же периоде/ambient **плоский** (bias +0.43→+0.84) → специфично
  для ГПА-1, **не** станционный сезонный эффект.
- **Класс:** (c)-кандидат (развивающееся состояние юнита), **не** подтверждённый дефект.
- **Оговорка:** НЕТ корроборации по КПД — остаточный дрейф `polytropic_head` незначим (z=−1.23).
  Может быть операционным (смена режима ГПА-1 после простоя) или сенсорным. Одно окно, один прогон.
- **Действие:** визуально посмотреть тренд `oil_temp_out_st` / маслосистемы ГПА-1 за 2-ю половину
  июня; сверить с журналом ТОиР — были ли работы / смена уставок на ГПА-1 в этот период.

## Сигнал 2 — ГПА-2: `vibro_rotor_front_horiz`

- **Что:** горизонтальная вибрация переднего ротора ГПА-2 недопокрывает коридор нормы
  (покрытие 0.875 при цели 0.98), нарушения с **восходящим трендом** (viol_z = 10.5).
- **Локализация:** ГПА-1 на этом датчике **чист** (0.983) → специфично для ГПА-2 (близнец-контроль
  на этом канале не сдвинут, поэтому сигнал законный).
- **Класс:** локализованный кандидат — раннее по динамике переднего ротора ГПА-2 **либо** пробел модели.
- **Оговорка:** один датчик, одно окно, побочный прогон.
- **Действие:** посмотреть тренд горизонтальной вибрации переднего ротора ГПА-2; сверить с осевым
  и вертикальным каналами того же узла (совпадает ли подпись).

## Сигнал 3 — ГПА-2: `gas_temp_out_gpa` (темп. нагнетания газа) + производный η_p

- **Что:** within-unit-декомпозиция (`_within_unit_bc.py`) на чистом окне GPA-2 (июнь, без
  экстраполяции) показала **прогрессирующую компоненту остатка −1.8σ_train** на температуре
  нагнетания газа, НЕ обратимую с ambient (rev=0.03 — не следует суточным/синоптическим колебаниям),
  и после вычитания ambient-связи (коллинеарность с сезоном разорвана). Эхом в `polytropic_head`
  (−1.8σ, вычисляется из темп. нагнетания) → один сигнал газового/компрессорного тракта.
- **Локализация:** per-unit guard 2/16 (12.5%) → локализовано, НЕ юнит-уровневый сдвиг. Остальные
  13 датчиков GPA-2 чисты или (b)-обратимы.
- **Класс:** (c)-кандидат (прогрессирующее, ambient-необратимое, локализованное).
- **Оговорка:** −1.8σ лишь чуть выше порога 1.5; gas_temp + η_p = один связанный сигнал (не два
  независимых); один юнит, одно окно. Самый строго-выведенный из трёх (прошёл разрыв коллинеарности
  + guard), но всё ещё «посмотрите сюда», не дефект. С сигналами 1/2 не корроборирует (разные узлы).
- **Действие:** посмотреть тренд температуры нагнетания газа GPA-2 и КПД (η_p) за июнь; сверить с
  состоянием проточной части/охлаждения; журнал ТОиР по GPA-2.
