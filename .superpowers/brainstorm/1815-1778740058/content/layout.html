Какой приоритет у информации?

Текущий дашборд показывает всё сразу — нужно выбрать главное. Какой сценарий ближе?

A

Операторский — "Всё ли в норме?"

Главный экран: 3 больших плитки ГПА со статусом. Детали скрыты до клика. Минимум текста, максимум цвета. Идеально для дежурного, который смотрит на экран раз в 5 минут.

B

Аналитический — "Что именно не так?"

Главный экран: список аномальных датчиков + график сразу открытый. ГПА-статус вторичен. Для инженера, который уже знает что есть проблема и хочет разобраться.

C

Гибрид — обзор + детали в одном экране

Верхняя полоса: ГПА-статус + счётчики. Левый сайдбар: только аномальные датчики (остальные скрыты). Правая часть: график. Упрощённая версия текущего, но без перегруза.
